## 项目背景

CodeFlow 是一款帮助小型工程团队简化 Code Review 流程的 SaaS 工具，解决传统 PR 审查中信息分散、效率低下的痛点。

## 我的角色

作为独立开发者，完成了产品设计、前端开发和 MVP 上线的全部工作，历时 6 周。

## 解决方案

将 GitHub PR、评论、任务清单整合进一个统一的 Dashboard，并加入 AI 辅助的代码摘要功能，帮助审查者快速理解改动意图。

## 成果

- Beta 上线后 3 周内获得 200+ 注册用户
- 团队平均 Review 时间缩短 40%
- Product Hunt 当日排名 Top 10

## 技术栈

- React 18 + TypeScript
- Next.js 14 App Router
- Tailwind CSS + Radix UI
- GitHub REST API
- Vercel 部署
