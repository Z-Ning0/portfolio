## 项目背景

Nexus Analytics 是一款面向早期初创公司的轻量级数据分析平台，帮助非技术团队快速理解业务数据，做出更明智的决策。

## 我的角色

负责产品的完整 UI/UX 设计以及前端开发，从 0 到 1 完成了从原型到上线的全流程。

## 解决方案

通过模块化的 Dashboard 设计，将复杂数据以卡片式可视化图表呈现，降低认知负担。

## 成果

- 日活跃用户从 200 增长至 2,000+
- 用户平均会话时长提升 45%
- App Store 评分 4.8 / 5.0

## 技术栈

- React 18 + TypeScript
- Recharts 数据可视化
- Tailwind CSS
- Vercel 部署
